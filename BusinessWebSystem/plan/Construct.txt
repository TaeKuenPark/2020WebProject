         	   -> 로그인실패 -> 경고창

2. 소개(intro) 
기본적인 내용(회사 소개),
 대표 전화번호( 상담전화안내 버튼누르면 상담안내 페이지로),
 길안내(상세 길안내) 누르면 길안내 페이지로 이동

길안내(introLoad)
주소와 네이버 네비게이션 서비스 연결

상담전화(introTelephone)
 대표전화, 사장님 연락처, 팩스 번호
 모바일에서는 전화 번호를 누르면 자동 연결

재고목록(stock)
 새제품(기본)(newly), 중고품(old)
  PC사진형 게시판, 글목록형 게시판 선택가능

 사진이 없는경우 ' 사진을 준비중입니다' 경고가 나오도록
 클릭해서 들어가면 외부사진, 내부사진,해당 목록이 나와있는 표(제품의 규격, 비고, 총금액)를 확인 가능
 새제품의 경우 '총금액'을 임대 개월당 금액, 
 주문시 제품 구매를 누르면전화 상담 안내를 안내해주는 창을 기본으로(모바일은 즉시 전화 연결) 
 중고품의 경우 판매상태(판매 완료, 재고 있음)
 판매 완료의 경우 1일 이상 노출되도록 설정

제작의뢰(order)
 현재 제품 제작 도면, 제작 상담을 올리는 기능
 의뢰 상태 : 의뢰중(아직 주문단계), 제작중(제품 주문제작 진행중), 제작보류, 제작완료(제품 제작완료- 글자색 다르게 노출), 판매완료- 제품 배송완료
 하단기능 - 주문제작, 의뢰 취소
	주문제작 - 제작의뢰자 명칭, 제작 의뢰자 전화번호, 제작의뢰 글 비밀번호, 의뢰 규격(필수) -> 글제목 : 금일날짜_의뢰자명
		       -  내용, 추가사항, 도면파일 첨부(권고)

	의뢰취소 - 제작의뢰 글 비밀번호, 관리자계정으로 주문제작에 올린글 삭제 가능
		        제작의뢰 글 비밀번호 인증후 삭제하기 전 경고나오도록 할것

A/S상담(AScenter)
 전화 상담이 기본원칙. 전화번호를 연결하는 페이지가 기본(ASTelephone)
 의뢰 상태 : AS요청 - A/S미진행 , AS진행중
 상담의뢰  - A/S요청작성  -  A/S의뢰자 명칭, A/S 의뢰자 전화번호, A/S의뢰 글 비밀번호, 의뢰 규격(필수) -> 글제목 : 금일날짜_의뢰자명
			A/S요청 사진 첨부 부분 추가
	  - A/S요청삭제  - A/S의뢰 글 비밀번호, 관리자계정으로 주문제작에 올린글 삭제 가능
		 	A/S의뢰 글 비밀번호 인증후 삭제하기 전 경고나오도록 할것

	 - 댓글로 A/S답글 달수있도록 제작

공지사항(notice)
 업데이트, 회사 공지사항 등록 페이지, 관리자가 관리

MYPAGE(M) 	- 스케줄러(scheduler)
		 	- 의뢰 현황 - 주문제작(위 주문제작 페이지 상태 관리)(orderStatus)
			 	   - A/S 현황(위 A/S상담 관리 페이지 상태 관리)(ASstatus)
		 	- 주요기능 - 국세청 세금계산서 웹이동(etc)
				  - 계약서 발행(pdf, excel 지원)
				  - 제작요금 계산할수 있는 계산 페이지
				  - 계약서 파일 열람(계약서 발행 버튼을 누르면 excel의 형태로 서버에 저장)
					- 계약서 Backup(월별로 나뉘어서 로컬 경로에 자동백업 기능)

			- 기능 메뉴얼 - 직원들이 볼수있는 홈페이지 기능 설명(menual)
